## ⚡ Avvio Rapido

```bash
# Installa dipendenze
npm install

# Configura variabili d'ambiente (vedi docs/SETUP.md)
cp .env.example .env

# Avvia l'applicazione
./manage.sh start
```

**🌐 Accesso**: http://localhost:3000

## 🎮 Caratteristiche

- � **Ricerca Pokemon** avanzata per nome, numero Pokedex, tipo
